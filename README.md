# 家庭旅行足迹地图 🧭

一个网页版的旅行打卡地图：在地图上标记去过的地方，记下日期和回忆。

## 怎么用（本地版）

直接**双击 `index.html`** 用浏览器打开即可（推荐 Chrome / Edge）。

- **搜地名**：右上角搜索框输入城市/地点名（如「东京塔」）→ 选中候选 → 地图飞过去并自动弹出表单、填好地名和坐标
- 或在地图上**点一下** → 填地点名称 / 日期 / 备注 → 保存，就多了一个打卡点 📍
- 添加时可选**上传 1 张照片**（本地版会自动压缩到约 1000px，省空间）；照片会显示在地图弹窗和左侧列表里
- 左侧列表点「定位」可飞到对应地点；点「删除」可移除
- **导出备份**：把所有打卡点存成一个 `.json` 文件，妥善保管
- **导入**：从备份文件恢复 / 合并打卡点

## 数据存在哪？

当前是 **本地版**：数据存在你这台电脑的浏览器里（localStorage）。

- 优点：零门槛，打开就能用，完全私密
- 注意：**换设备看不到、清浏览器缓存会丢** → 所以请定期用「导出备份」存一份

## 技术栈

- 地图：[Leaflet](https://leafletjs.com/) + OpenStreetMap（免费、无需密钥）
- 纯原生 HTML / CSS / JavaScript，无框架

## 本地开发怎么跑

OAuth 必须通过 http(s) 访问，不能直接双击打开。在 `map-checkin` 目录下运行：

```
python -m http.server 8000
```

然后浏览器打开 **http://localhost:8000** （注意必须是 localhost，不能用 127.0.0.1，要和 Google 后台登记的来源一致）。
